## 提出方法

下記コマンドを実行すると、正しくコンパイル・実行可能なコードの場合に自動で採点が開始される

```bash

```

下記コマンドを実行すると、正しくコンパイル・実行可能なコードで正しい出力をするかを確認出来る

```bash
./test_all.sh
```
